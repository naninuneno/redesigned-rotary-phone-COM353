Terminology learned:
Looping Statement
Used to perform tasks repeatedly
Loop Control
Specified what condition is necessary for the continued execution of the loop
Loop Body
Statements that will be repeatedly executed as long as the loop control condition is satisfied
For loop
A loop that uses a counter, a test condition and a way to update the counter. Useful for when the value of the current iteration is important / you know how many iterations are required beforehand e.g. looping through an array so you know to stop when the counter reaches the array's length
While loop
A loop that only uses a test condition, which relies on the loop body being able to eventually satisfy the condition so that the loop is terminated. Useful for when the number of iterations is not known beforehand
Do... While loop
A loop that works similar to a while loop, except the looping condition is checked after the loop instead of before, so it behaves just like a while loop except it will always run at least once (before the condition is tested at the end of the loop)
Associative Array
Uses a string as an index e.g. Course["ICT"] = "Information Communication Technology";

Practical 7:

1. Explaining Javascript statements:

  • var myAge = 45;

    Declaring a variable 'myAge' and assigning it a value of 45. myAge's value == 45

  • myAge += 4;

    Adding 4 to myAge - shorthand for 'myAge = myAge + 4'. myAge's value == 49

  • myAge++;

    Increment myAge by 1 - shorthand for 'myAge = myAge + 1'. myAge's value == 50

  • myAge == 40 + 10;

    Boolean expression, an equality check for myAge comparing it to 50. Evaluates to true

  • myAge <= 38;

    Boolean expression, checking that myAge is equal to or less than 38. Evaluates to false

  • myAge = 40;

    Assigning myAge a value of 40
